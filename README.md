# goptional

[![GoDoc][doc-img]][doc] [![Build Status][ci-img]][ci] [![Coverage Status][cov-img]][cov]

A generic-based implementation of the `Optional` type in Go

## Installation

```bash
go get -u github.com/nykolynoleg/goptional
```
